PROGRAM.PY USO:
Linguagem: Python 3.x
Bibliotecas utilizadas:

uuid — geração de IDs únicos para usuários
math (implícito) — operações aritméticas nativas

Funcionalidades implementadas:

Menu interativo via terminal (loop principal)
Tratamento de erros com try/except
Módulo Contador — cálculo de lucro, soma e divisão
Módulo Banco — cadastro de usuário, depósito, saque e transferência
Controle de fluxo com while True e break

Conceitos aplicados:

Tipagem e conversão de tipos (int, float, str)
Variáveis de controle de estado (bv para sessão do banco)
Arredondamento de valores monetários com round()
